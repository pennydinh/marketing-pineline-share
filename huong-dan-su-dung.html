1. Yêu Cầu Trước Khi Bắt Đầu

  • Máy tính Mac hoặc Windows (đã cài Node.js)
  • Tài khoản Anthropic (có API Key – để AI viết bài)
  • Tài khoản OpenAI (có API Key – để AI tạo ảnh)
  • Tài khoản Facebook Page (có Access Token)
  • File cookies.json (Cookie đăng nhập Facebook cho Bot đăng Groups)
  • Kết nối Internet ổn định
💡 Mẹo: Tất cả API Key đã được cài sẵn trong file .env.local. Bạn chỉ cần thay đổi khi key hết hạn hoặc hết tiền.

2. Cài Đặt Lần Đầu (Chỉ Làm 1 Lần Duy Nhất)

1 Tải source code về máy
Giải nén file ZIP hoặc clone từ Github
git clone https://github.com/mp1391004/ai-content-pipeline.git
cd ai-content-pipeline
2 Cài đặt thư viện cho Web
Cài tất cả các package cần thiết
npm install
3 Cài đặt thư viện cho Bot
Cài riêng cho phần Bot làm video + đăng nhóm
cd bot
npm install
cd video-maker
npm install
4 Cấu hình API Keys
Mở Terminal, gõ lệnh sau để tạo và chỉnh sửa file chứa API Keys
# Bước 1: Vào thư mục dự án
cd ai-content-pipeline

# Bước 2: Mở file .env.local bằng trình soạn thảo nano
nano .env.local

# Bước 3: Dán nội dung dưới đây vào, thay xxxxx bằng key thật của bạn:
ANTHROPIC_API_KEY=sk-ant-xxxxx
OPENAI_API_KEY=sk-proj-xxxxx
DATABASE_URL=postgresql://xxxxx
FACEBOOK_ACCESS_TOKEN=EAAxxxxx
FACEBOOK_PAGE_ID=1234567890
RAPID_API_KEY=xxxxx
BRAVE_API_KEY=xxxxx
# Bước 4: Lưu file
# Bấm Ctrl + O (lưu) → Enter → Ctrl + X (thoát)

# Hoặc nếu bạn quen dùng VS Code, gõ lệnh này thay cho nano:
code .env.local
5 Deploy Web lên Vercel
Đẩy code lên Github, Vercel sẽ tự động build
git add .
git commit -m 'Initial deploy'
git push
✅ Hoàn tất cài đặt! Từ giờ trở đi, bạn chỉ cần làm theo Quy Trình Hàng Ngày ở phần tiếp theo.

3. Quy Trình Sử Dụng Hàng Ngày

Mỗi ngày, bạn chỉ cần làm 2 việc:

ViệcHành độngỞ đâuMất bao lâu
Việc 1 🤖 Bật Bot (chỉ cần gõ 2 dòng lệnh) Terminal máy tính 30 giây
Việc 2 🌐 Lên Web tạo bài, chọn ảnh, hẹn giờ đăng Trình duyệt (điện thoại/laptop) 10 phút

4. Bước 1: Research – Tìm Kiếm Tin Tức

1 Mở trang Web
Truy cập URL Vercel của bạn (ví dụ: ai-content-pipeline-mp.vercel.app)
⬇️
2 Chọn nguồn tin muốn cào
Tick chọn các nguồn tin tức bạn muốn hệ thống quét tự động:
• Brave Search – Tin tức tổng hợp
• X/Twitter – Bài trending trên MXH
• Instagram – Bài viral trên IG
• TechCrunch / a16z / IndieHackers – Tin chuyên ngành startup & AI
⬇️
3 Bấm nút "Tìm kiếm"
Hệ thống sẽ tự động quét tất cả các nguồn đã chọn và hiển thị danh sách tin nóng mới nhất. Chờ 15-30 giây.
💡 Mẹo: Không cần nhập từ khoá! Hệ thống sẽ tự động cào những bài mới nhất, trending nhất từ các nguồn bạn đã tick.

5. Bước 2: Lọc & Chọn Format Viết Bài

1 Tick chọn các bài hay
Đọc tiêu đề và tóm tắt, tick vào ô checkbox những bài bạn muốn viết lại
⬇️
2 Chọn Format cho từng bài
Mỗi bài có 2 lựa chọn:
• 📝 POV – Bài phân tích chuyên sâu, viết theo góc nhìn của Phương
• ⚡ TIN NHANH – Bài tin ngắn gọn, giật gân, dạng "Breaking News"
⬇️
3 Bấm nút "✍️ Viết & Tạo ảnh cho X bài đã chọn"
AI sẽ tự động:
• Claude Opus 4.5 viết bài theo persona Phương
• DALL-E 3 tạo ảnh minh hoạ phong cách Cinematic/Ghibli
⏱️ Chờ khoảng 30-60 giây mỗi bài
⚠️ Lưu ý: Nếu thấy thông báo "CẢNH BÁO LỖI TỪ AI", nghĩa là API key hết tiền hoặc hết quota. Kiểm tra lại tài khoản Anthropic/OpenAI.

6. Bước 3: Review, Chọn Ảnh & Lên Lịch Đăng

1 Đọc lại bài viết
Sửa nội dung nếu cần (bấm vào ô text để chỉnh sửa). Sửa hashtag nếu muốn.
⬇️
2 Chọn ảnh minh hoạ
Mỗi bài có 2 ảnh để chọn:
• Ảnh Báo Gốc – ảnh từ bài viết nguồn
• Ảnh AI Tạo – ảnh do DALL-E 3 vẽ
Bấm vào ảnh muốn dùng, dấu ✓ xanh sẽ hiện lên.

• Bắt đầu đăng từ (Giờ) – chọn ngày giờ bài đầu tiên lên sóng
• Khoảng cách mỗi bài (Tiếng) – ví dụ: 3 = mỗi 3 tiếng đăng 1 bài
• ☑️ Tự động tạo & Đăng Video Reels – tick nếu muốn Bot làm video
⬇️
4 Tick chọn các bài muốn đăng
Tick vào ô checkbox bên trái các bài đã ưng ý
⬇️
5 Bấm nút Đăng
Có 3 lựa chọn:

NútTác dụng
📅 Đăng PageChỉ đăng lên Facebook Page (theo lịch đã hẹn)
👥 Đăng Hội NhómChỉ gửi lệnh cho Bot đăng vào các Groups
🚀 Đăng Tất CảĐăng Page + Đăng Groups + Tạo Video (nếu tick)
🎯 Khuyên dùng: Bấm "🚀 Đăng Tất Cả" cho nhanh gọn! Hệ thống sẽ tự lên lịch Page đúng giờ và gửi lệnh cho Bot xử lý phần còn lại.

8. Câu Hỏi Thường Gặp (FAQ)

❓ Tôi có thể dùng trên điện thoại không?

✅ Có! Phần Web (tạo bài, chọn ảnh, hẹn giờ) hoạt động hoàn hảo trên điện thoại. Chỉ riêng phần Bot thì cần máy tính để chạy.

❓ Bot có chạy khi tôi không ở nhà không?

✅ Có, miễn là máy tính vẫn bật và Terminal không bị tắt. Bạn có thể lên Web bằng điện thoại ở bất kỳ đâu, tạo bài và hẹn giờ. Bot ở nhà sẽ tự làm.

❓ Tôi tick 5 bài làm video, nó làm 1 bài rồi nghỉ?

❌ Không! Bot sẽ làm liên tục hết 5 bài (cuốn chiếu) rồi mới nghỉ chờ lệnh mới.

❓ Bài hẹn 3h chiều, Bot có đăng nhóm trước 3h không?

❌ Không. Bot chỉ đăng nhóm khi đã tới giờ hẹn. Trước đó nó sẽ bỏ qua và quét lại sau 10 phút.

❓ Video Reels có lên đúng giờ tôi hẹn không?

✅ Có. Nếu bạn set lịch, video sẽ được upload ngay nhưng Facebook sẽ giữ và công bố đúng giờ hẹn.

❓ Tôi muốn thay đổi danh sách nhóm Facebook?

Mở file bot/post-groups.js, tìm biến GROUPS ở đầu file, thêm/xoá URL nhóm.

❓ Cookie Facebook hết hạn thì sao?

Mở trình duyệt Chrome, đăng nhập Facebook, dùng extension xuất cookies ra file cookies.json, thay thế file cũ trong thư mục bot/.

9. Xử Lý Sự Cố

Lỗi gặp phảiNguyên nhânCách khắc phục
Web báo "CẢNH BÁO LỖI TỪ AI" API key hết tiền hoặc quota Vào console.anthropic.com hoặc platform.openai.com kiểm tra credit, nạp thêm tiền
Web báo "404 model not found" Model AI bị giới hạn Liên hệ người phát triển để đổi model trong code
Terminal báo "MODULE_NOT_FOUND" Chạy node post-groups.js sai thư mục Phải gõ cd bot trước, rồi mới gõ node post-groups.js
Bot báo "cookies lỗi" Cookie Facebook đã hết hạn Xuất lại cookies.json từ trình duyệt Chrome
Video không render được Thiếu thư viện Remotion Gõ: cd bot/video-maker && npm install
Facebook chặn đăng nhóm Đăng quá nhanh / IP bất thường Chờ 24 giờ rồi thử lại. Không đăng quá 10 nhóm/ngày
Giao diện điện thoại bị vỡ Cache trình duyệt cũ Xoá cache: Settings → Safari → Clear Cache. Hoặc dùng Chrome bấm Ctrl+Shift+R
Web không load được Vercel chưa build xong Chờ 2 phút sau khi push code, rồi F5 lại
