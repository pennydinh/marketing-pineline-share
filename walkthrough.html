🔄 Luồng Hoạt Động (End-to-End)

1️⃣
👤 Penny mở Web → Bước 1: Research
Nhập keyword, chọn nguồn (TechCrunch, X, Instagram...), bấm "Tìm kiếm"
⬇️
2️⃣
⚡ API cào tin
Firecrawl bóc HTML báo chí → RapidAPI kéo bài MXH trending
⬇️
3️⃣
📋 Bước 2: Lọc & Chọn Format
Penny tick bài hay, chọn POV (góc nhìn) hoặc Tin Nhanh (giật gân)
⬇️
4️⃣
✍️ Claude Opus 4.5 viết bài + 🎨 DALL-E 3 tạo ảnh
AI viết content theo persona Phương, đồng thời tạo ảnh 1:1 Cinematic/Ghibli
⬇️
5️⃣
📋 Bước 3: Review & Lên Lịch
Penny sửa bài nếu cần, chọn ảnh (gốc hoặc AI), đặt giờ đăng, tick "Tạo Video"
⬇️
6️⃣
🚀 Penny bấm "Đăng"
• Đăng Page → Facebook Graph API lên lịch / đăng ngay
• Tạo Video → Lệnh gửi xuống DB, Bot tự nhặt
• Đăng Groups → Lệnh gửi xuống DB, Bot chờ đúng giờ rồi đăng
⬇️
7️⃣
🤖 Bot Local (Tự động)
• Gemini viết kịch bản → TTS → Remotion render video → Upload Reels
• Playwright mở Chrome ảo → Đăng bài + ảnh vào 4 nhóm FB
⬇️
✅
Hoàn tất!
DB cập nhật trạng thái → Web hiển thị Lịch sử hoạt động
